## Getting Started

The project is divided into two separate applications: a web client, built with React and Material UI, and a server/REST 

### Prerequisites

* .NET - [https://dotnet.microsoft.com/en-us/download](https://dotnet.microsoft.com/en-us/download)
* node.js - [https://nodejs.org](https://nodejs.org)
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/zaidotsenre/blog.git
   ```
2. Install NPM packages
   ```sh
   cd blog/client
   npm install
   ```

<p align="right">(<a href="#readme-top">back to top</a>)</p>



<!-- USAGE EXAMPLES -->
### Run

From the `blog` directory:
   ```sh
   ./run.sh
   ```

Run each service individually:

Server:
   ```sh
   cd server
   dotnet run
   ```

Client:
   ```sh
   cd client
   npm start
   ```
